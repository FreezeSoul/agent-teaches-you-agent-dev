# Manus AI 收购被阻止：地缘政治如何封闭了一条 Agent 技术独立路线

> **摘要**：2026年4月27日，中国国家发改委正式阻止 Meta 对 Manus AI 的 20亿美元收购案，要求"解除交易"。这不仅是本季度最大的 AI 收购事件，更揭示了一个深刻的结构性矛盾：当一家中国创始团队公司通过"新加坡跳转"试图进入美国生态，技术原产地的地缘属性并不会因此消失。本文分析这一事件对 Agent 技术生态、跨境资本流动、以及 Manus engram 技术独立发展路线的影响。

## 一、事件经过：20亿美元的收购如何走向解除

### 时间线

| 时间 | 事件 |
|------|------|
| 2022年 | Manus AI 在北京创立，后将总部迁至新加坡 |
| 2025年12月 | Meta 宣布以超过 20亿美元收购 Manus AI |
| 2026年1月 | 中国国家发改委（NDRC）启动调查；两名联合创始人（Xiao Hong、Ji Yichao）被禁止出境 |
| 2026年1-4月 | 审查期间，Meta 已将 Manus 整合进内部系统，Manus 高管加入 Meta |
| 2026年4月27日 | NDRC 正式阻止交易，要求 Meta"解除收购" |

### 关键细节

**交易结构**：据多方报道，该交易包含 5亿美元的员工留存池，Meta 在宣布收购时已让 Manus 高管加入公司，这一操作使局面复杂化。

**中国的核心关切**：NDRC 的审查聚焦于三个问题：
1. 从中国大陆向新加坡转移人员和 AI 系统是否获得了必要的出口许可
2. 是否违反了技术出口管制法规
3. 是否涉及数据跨境传输

**Meta 的处境**：CNBC 报道 Meta 表示"完全遵守适用法律"，并暗示正在评估包括向特朗普政府求助在内的所有选项。但分析普遍认为逆转的窗口极小。

## 二、"新加坡跳转"模式的结构性失效

### 什么是"新加坡跳转"

过去几年，大量中国科技创业者将公司总部迁至新加坡，声称自己是"新加坡公司"以吸引西方资本并在监管上获得更宽松的待遇。这种做法在创投圈被称为 **"Singapore Washing"**。

Manus 是这一模式的典型案例——在北京创立，迁至新加坡，宣称自己是国际创业公司，然后被 Meta 收购。

### 这次为什么失效了

NDRC 用行动证明：** incorporation location 不等于技术原产地**。即使 Manus 法律上已是新加坡公司，只要核心技术和人员来源可以追溯到中国大陆，中国监管机构就认为自己有管辖权。

这是中国出口管制框架的一次重要宣示：技术不会因为被装进新加坡壳就"脱敏"。

### 地缘背景

阻止收购的时间点（4月27日）恰逢特朗普与习近平预期会面之前。这个时机不是偶然的——北京在高层对话前夕展示其在关键技术上不会退让的姿态。

## 三、Manus 的 engram 技术与 DeepSeek Engram 的根本区别

> ⚠️ **重要澄清**：本文必须明确区分两个名为 "engram" 的技术，它们之间没有直接关系。

### Manus AI 的 engram

Manus AI 的 "engram" 是其 **General Purpose Agent 架构的核心组件**，指的是一种条件性记忆机制——让 Agent 能够根据当前任务上下文有条件地调用历史经验，从而实现更高效的少样本学习和持续适应。

Manus 在 GAIA（General AI Assistants）benchmark 上达到 **86.5% pass@1**，超越 OpenAI Deep Research，展现了这一技术路线在通用任务处理上的竞争力。

### DeepSeek Engram

DeepSeek 的 Engram（arXiv:2601.07372，2026年1月）是一个**模型架构层面的条件记忆模块**，作为 Transformer 的附加组件，通过 N-gram 哈希查找将静态知识卸载到外部记忆，让 GPU 专注推理计算。这是一个底层的模型效率优化技术，已在 repo 的 `articles/frameworks/deepseek-engram-conditional-memory-2601-07372.md` 中有详细解析。

### 总结对比

| 维度 | Manus engram | DeepSeek Engram |
|------|-------------|-----------------|
| **技术层级** | Agent 系统架构 | 模型权重/架构模块 |
| **功能** | Agent 持续学习/条件性经验调用 | 静态知识存储与 O(1) 查找 |
| **与 Transformer 的关系** | Agent 的外部组件 | Transformer 的内部组件 |
| **开源状态** | 未开源 | 已开源（GitHub: deepseek-ai/Engram） |
| **与 Meta 的关系** | Meta 收购对象，被阻止 | DeepSeek 独立研究 |

## 四、Manus engram 技术独立发展的可能路径

收购被阻止后，Manus 的 engram 技术面临三条路径：

### 路径一：Meta 保留知识产权，继续独立研发

CNN 报道 Meta 已在调查期间将 Manus 整合进内部系统。如果 Meta 选择抗辩（比如通过美国政府渠道），可能在技术层面继续推进，但政治风险极高。

**可能性**：中（受外交层面影响）

### 路径二：Manus 退回独立运营，但受制于创始人的出境限制

TechCrunch 指出两名联合创始人 Xiao Hong（CEO）和 Ji Yichao（首席科学家）被禁止离境。这意味着他们无法赴美参与 Meta 的工程协作，实质上切断了与 Meta 的深度整合。

Manus 可能退回新加坡，作为一个独立公司继续运营，但其核心竞争力（创始团队与 Meta 的深度绑定）已被削弱。

**可能性**：高

### 路径三：中国资本接盘，engram 技术成为中国 AI 生态的一部分

如果 Meta 无法保留 Manus，最直接的结果是 Manus 被迫寻找其他资金来源或买家。中国本土资本或国有企业可能成为接盘方，使 engram 技术进入中国 AI 生态体系。

**可能性**：中（取决于北京的后续态度）

### 核心不确定性

Manus 的 engram 技术能否独立发展，关键不在技术本身，而在于：
1. 创始团队能否获得出境许可参与国际协作
2. 是否有足够的资金维持独立研发
3. 中国是否允许其继续与外资生态保持距离

## 五、对 Agent 技术生态的深层影响

### 对"出海"创业模式的打击

Manus 案例证明，"新加坡跳转"对战略性 AI 技术不再有效。这将迫使中国 AI 创业者重新评估跨境融资和退出的风险结构。

### 对美国科技巨头的教训

Meta 以超过 20亿美元收购一家"新加坡公司"，结果被中国监管机构正式阻止——这是美国科技公司在中国地缘政治风险评估上的重大失算。后续美国公司在评估收购中国背景的技术公司时会更加谨慎。

### 对 Agent 架构竞争格局的影响

Manus 在 GAIA benchmark 上 86.5% 的表现说明，基于 engram 条件记忆技术的架构在通用 Agent 任务上有真实的竞争优势。如果 Manus 因地缘政治被迫降速，这一技术路线上的潜在竞争者（尤其是 OpenAI Deep Research 等）将获得更大的相对优势。
