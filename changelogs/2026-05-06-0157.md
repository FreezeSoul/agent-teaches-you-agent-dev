### `articles/context-memory/anthropic-cursor-token-efficient-compaction-2026.md`

**注意力预算与 Token 高效压缩：Anthropic 和 Cursor 共同指向的长程 Agent 进化方向**

核心论点：长程 Agent 的核心挑战不是「扩大上下文窗口」，而是如何在有限注意力预算内做出「压缩什么、保留什么」的高质量决策。Anthropic 的注意力预算理论与 Cursor 的自概括训练（Compaction-in-the-Loop）从不同角度得出一致结论——未来的长程 Agent 必须具备「learned context compression」能力，而非依赖更大的上下文窗口。

关键洞察：
- **注意力预算是结构性的有限资源**：Transformer 的 n² 配对关系导致上下文越长，有效信息密度越低
- **System Prompt 的 Goldilocks Zone**：既不过度工程化（if-else brittle），也不过度抽象（无具体输出信号）
- **Compaction-in-the-Loop 的训练范式**：压缩质量应作为训练目标，通过 reward signal 同时优化 agent 行为和压缩质量
- **两者的技术收敛**：Anthropic（工程实践：curation） + Cursor（训练方法：learned compression） → 同一方向
