### 三大核心原则

1. **保持简单**：最成功的实现不是来自复杂框架，而是来自简单可组合的模式
2. **优先透明度**：显式展示 Agent 的规划步骤
3. **精心设计 ACI**：Agent-Computer Interface，通过工具文档和测试精心设计

### Anthropic 的 Agent 定义

> Agent = LLM 动态指挥自己的进程和工具使用，在循环中自主运行

**对比 Workflow**：
- Workflow = 预定义代码路径
- Agent = LLM 动态决定
